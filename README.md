# DevOps Demo POC (Notes API)

Minimal project for DevOps beginners:
- Java 17 + Spring Boot (REST)
- PostgreSQL
- Docker + docker-compose
- Kubernetes manifests (kind/minikube)
- GitHub Actions (build/test + push image)

Endpoints:
- GET  /api/notes
- GET  /api/notes/{id}
- POST /api/notes
- DELETE /api/notes/{id}
